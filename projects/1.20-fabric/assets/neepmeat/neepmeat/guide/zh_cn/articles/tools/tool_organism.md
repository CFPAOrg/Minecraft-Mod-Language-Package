# 工具生物体

工具生物体会为工具提供自我修复功能。

可使用PLC的COMBINE操作将其嵌装至任意具有耐久度的物品。

它刻替代钻具底盘中的气动发动机，可令其在周围没有压缩机的情况下运作。

# 补充消耗品

工具生物体的运作依托于代谢底物，如肉浆、动物饲料、液态食物等。可在灌装机中进行补充，也可右击储罐。

# 能量值

- 动物饲料：每桶500耐久度
- 肉浆：每桶750耐久度
- 液态食物：81000 * 饥饿值 / 600
