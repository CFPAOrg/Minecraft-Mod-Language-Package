# Round Robin Item P2P Tunnel

The Round Robin Item P2P Tunnel part ensures that items entering your ME network are spread evenly across all connected outputs, even when you insert items multiple times. Unlike the standard Item P2P Tunnel, which always sends to the nearest output first, this version tracks usage and directs each batch to the output that hasnâ€™t received anything for the longest time.
