# 内存卡

<ItemImage id="memory_card" scale="4" />

内存卡可在AE2[设备](../ae2-mechanics/devices.md)间复制粘贴设置，也可连接[P2P通道](p2p_tunnels.md)。它也能将[升级卡](upgrade_cards.md)复制到设备中。

- Shift右击以生成新P2P连接频率。
- 右击以粘贴设置或升级卡，或连接频率。
