# Halberd

\columns[fit=second]{The halberd is a module for the Staff base. It has a number of different melee attacks.
}{\item_render{meatweapons:halberd}}

## Attacks

When unmounted:

- Attack 1: Charges if held while sprinting. When released, the user leaps forwards a number of blocks, damaging entities in the way.
    - Base damage: 8
- Attack 2: Attacks the targeted entity. 
    - Base damage: 7
    - Range: 3

When mounted:

- Attack 1: Hooks and grabs the targeted entity, dragging it along for about a second.
- Attack 2: Same as when unmounted.
