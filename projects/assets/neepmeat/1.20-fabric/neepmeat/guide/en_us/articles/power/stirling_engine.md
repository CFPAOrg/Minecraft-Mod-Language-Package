# Stirling Engine

\columns[fit=second]{The Stirling Engine converts heat energy from burning fuel into mechanical work to power motorised blocks. Unlike the other motors, it requires almost no setup, so it is useful for running a single machine away from a power system.
}{\item_render{neepmeat:stirling_engine}}

## Usage

The Stirling Engine can accept any valid furnace fuel and will rotate as long as the fuel burns. It delivers 150eJ/t to the connected load.

As with the other motors, a Stirling Engine will automatically orient itself towards a motorised block if placed next to it.
