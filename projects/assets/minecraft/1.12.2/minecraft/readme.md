# 全角字符修正与特殊字符补充
- 本包添加了几个特殊字符字体，用于补充部分新确定的元素名称
- 本包修正了中文全角字符的宽度和位置问题。

## Mapping
本包建议将下列生字采用左侧书写，在打包阶段使用脚本转换成右侧字符

| 翻译文本 |       转换字符       |                          备注                           |
| :------: | :------------------: | :-----------------------------------------------------: |
| [[钅卢]] |       `\ue900`       |                                                         |
| [[钅杜]] |       `\ue901`       |                                                         |
| [[钅喜]] |       `\ue902`       |                                                         |
| [[钅波]] |       `\ue903`       |                                                         |
| [[钅黑]] |       `\ue904`       |                                                         |
| [[钅麦]] |       `\u9fcf`       |                                                         |
| [[钅达]] |       `\ue906`       |                                                         |
| [[钅仑]] |       `\ue907`       |                                                         |
| [[钅哥]] |       `\u9fd4`       |                                                         |
| [[钅尔]] |       `\u9fed`       |                                                         |
| [[钅夫]] |       `\ue90a`       |                                                         |
|    镆    |       `\u9546`       |          此元素名已存在对应字符，无需修改替换           |
| [[钅立]] |       `\ue90c`       |                                                         |
| [[石田]] |       `\u9fec`       |                                                         |
| [[奥气]] |       `\u9feb`       |                                                         |
|    ……    |    `\ue908\ue909`    |                    全角省略号的修改                     |

旧版字符渲染只支持第一平面，后面更新支持第二平面，第三平面的了。
1.16 以后转换的是对应的字符
