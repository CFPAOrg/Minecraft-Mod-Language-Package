## 使用说明
本包为1.10.2相关热门模组的汉化资源包，使用和普通资源包使用方式相同    
主要面向FTB整合包模组汉化补全，鉴于很多模组作者既不开仓库，也不接受汉化，故此资源包加载方式为上策    
如果缺少某些模组汉化，请在issue反馈，如果你想提交部分汉化，欢迎发PR。
